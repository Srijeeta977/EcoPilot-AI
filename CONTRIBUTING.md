# Contributing to EcoPilot AI

Thank you for your interest in contributing to EcoPilot AI.

We welcome contributions that improve sustainability awareness, user experience, performance, accessibility, and code quality.

## Getting Started

1. Fork the repository
2. Clone your fork locally
3. Create a feature branch
4. Make your changes
5. Test your changes
6. Submit a Pull Request

## Development Guidelines

* Follow existing project structure
* Keep code modular and reusable
* Use meaningful commit messages
* Maintain responsive design standards
* Ensure accessibility considerations are respected

## Pull Request Process

1. Update documentation where necessary
2. Ensure the project builds successfully
3. Verify functionality before submission
4. Provide a clear description of changes

## Reporting Issues

When reporting issues, include:

* Expected behavior
* Actual behavior
* Steps to reproduce
* Screenshots if applicable

Thank you for helping improve EcoPilot AI.
